心がけているのは
「お客様のため」のWeb制作です。

Webサイトは集客し、売上を出すためのツール。

コーダーは「デザインカンプを忠実に再現するもの」ですが、Webサイトの本質は集客です。
ですので、私からも意見や疑問点は積極的に発言させていただきたいと思っております。

一緒にお仕事させていただく以上、パートナーとしてより結果を残せるWebサイトを構築いたします。

Skills

スキルと使用ツールの一覧です。

  • HTML

    適切なタグ、WAI-ARIAを用いてアクセシビリティを考慮したマークアップを意識しています。

  • CSS

    「良いCSS」に基づき命名規則(BEM)を用いたCSS設計で、運用しやすいサイトを構築します。

  • Saas

    Sassの機能である変数やmixinを利用することで、効率的にコーディングしています。

  • JavaScript

    ハンバーガーメニューやスクロールバーなどよくあるアニメーションの実装が可能です。

  • jQuery

    ハンバーガーメニューやスクロールバーなどよくあるアニメーションの実装が可能です。

  • PHP

    静的サイトをWordPress化するときに用います。

  • WordPress

    WordPressのテンプレートタグ、プラグインを用いて静的サイトをWordPress化します。

  • Gulp

    画像、ファイルの圧縮・コンパイルを自動化しているため、効率的にコーディングできます。

  • Git

    個人のフォルダをGitで管理しています。Gitの概念を理解しているので共同開発も可能です。
